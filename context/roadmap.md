# Roadmap del Proyecto (Scrum)

Este documento sirve como registro vivo de avances y tareas. El objetivo es
mantener visibilidad del trabajo y apoyar el aprendizaje.

## Backlog del Producto

- Reportar ruta (alta)
- Consultar "ultimo visto" (alta)
- Ver mapas (media)
- Moderacion / anti-spam (media)
- Panel o export de reportes (baja)

## Sprint 0 (Fundamentos)

- Definir el diagrama de estados del chatbot
- Definir el modelo de datos (entidades y relaciones)
- Definir reglas de expiracion de reportes
- Elegir stack y convenciones de proyecto

## Sprint 1 (Flujo basico)

- Implementar el menu principal
- Implementar flujo de reportar ruta
- Implementar flujo de "ultimo visto"
- Pruebas manuales con datos simulados

## Sprint 2 (Calidad y seguridad)

- Votaciones/confirmaciones
- Anti-spam basico
- Manejo de errores y logs
- Ajustes de UX conversacional
